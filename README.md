# 🏦 Banking Management System

## 📌 About Project
This is a Banking Management System developed using Python and Streamlit.  
It allows users to manage bank accounts and perform transactions easily.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- View Account Details
- Dashboard Interface

## 🛠️ Tech Stack
- Python
- Streamlit

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the project:
   streamlit run pro_dashboard.py

## 📂 Project Structure
- bank.py → backend logic
- dashboard.py / pro_dashboard.py → frontend
- requirements.txt → dependencies
